# NexWell Health Partners Landing Page Build Prompt

## Project Overview

Build a modern, conversion-focused landing page for NexWell Health Partners, a Management Services Organization (MSO) that provides operational, financial, and technology services to specialty medical practices, connecting them with hospitals and Federally Qualified Health Centers (FQHCs). The organization specializes in telehealth and telemedicine integration.

---

## Brand Identity

### Logo Reference
The NexWell logo features interconnected nodes forming an abstract "N" shape, symbolizing network connectivity and partnership. The design communicates integration, collaboration, and modern healthcare solutions.

### Color Palette

| Color | Hex Code | Usage |
|-------|----------|-------|
| Primary Teal | #1F7A8C | Headers, CTAs, primary elements |
| Deep Teal | #134B54 | Footer, text, dark backgrounds |
| Accent Gold | #C9A227 | Highlights, icons, accent buttons |
| Light Gold | #E8D5A3 | Subtle backgrounds, borders |
| White | #FFFFFF | Backgrounds, text on dark |
| Light Gray | #F5F7F9 | Section backgrounds |
| Dark Text | #2D3748 | Body copy |

### Typography

- **Headings**: Inter, Montserrat, or Poppins (600-700 weight)
- **Body**: Inter or Open Sans (400-500 weight)
- **Accent/CTAs**: Same as headings

### Visual Style Guidelines

- Clean, minimalist design with ample white space
- Subtle gradients (teal to deep teal)
- Abstract geometric patterns using node/connection motifs
- Professional medical imagery showing technology integration
- Rounded corners (8-12px) on cards and buttons
- Soft shadows for depth (0 4px 20px rgba(31, 122, 140, 0.1))

## Technical Requirements

### Framework Recommendations
- React/Next.js for dynamic components
- Tailwind CSS for styling
- Framer Motion for subtle animations

### Responsive Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Performance
- Lazy load images
- Compress all assets
- Target < 3s load time
- Core Web Vitals optimized

### Accessibility
- WCAG 2.1 AA compliant
- Semantic HTML
- Keyboard navigation
- Proper contrast ratios
- Alt text for all images

### SEO
- Meta title: "NexWell Health Partners | MSO Services for Specialty Practices"
- Meta description: "NexWell Health Partners provides management services, telehealth integration, and hospital partnerships for specialty medical practices. Focus on patients. We handle the business."
- Structured data for LocalBusiness
- Open Graph tags for social sharing

---

## Animation Guidelines

- Hero section: Subtle fade-in on load (0.5s)
- Stats counters: Animate numbers on scroll into view
- Service cards: Gentle hover lift (transform: translateY(-4px))
- Process timeline: Sequential reveal as user scrolls
- Testimonial carousel: Smooth crossfade transitions
- CTA buttons: Subtle color shift on hover

## Copy Tone Guidelines

- Professional but approachable
- Direct and clear—avoid jargon
- Benefits-focused, not feature-dumping
- Empathetic to physician pain points
- Confident without being aggressive
- Use "we" and "you" to create partnership feel

---

## Conversion Goals

1. Primary: Form submission for consultation
2. Secondary: Phone call
3. Tertiary: Resource download (future: gated whitepaper)

---

## Build Notes

- Use the NexWell logo as the definitive brand anchor
- The connected nodes icon should inspire subtle pattern elements throughout
- Teal is the dominant brand color; gold is for accents and CTAs
- Prioritize clarity over cleverness—healthcare executives want substance
- Mobile experience must be equally polished
- Ensure all forms are connected to CRM/email automation
